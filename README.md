# ConfigScan

ConfigScan est une application Windows développée en Python qui permet d'analyser et de scanner les configurations système.

## 📋 Description

ConfigScan est un outil de diagnostic et d'analyse des configurations système Windows. L'application fournit une interface utilisateur intuitive pour examiner les paramètres système, les configurations réseau, et autres éléments critiques de votre environnement Windows.

## ✨ Fonctionnalités

- 🔍 **Scan de configuration système** : Analyse complète des paramètres système Windows
- 🖥️ **Interface utilisateur moderne** : Interface graphique intuitive et responsive
- 📊 **Rapports détaillés** : Génération de rapports d'analyse complets
- ⚡ **Performance optimisée** : Scan rapide et efficace
- 🔒 **Sécurité** : Analyse sécurisée des configurations sans modification du système

## 🚀 Installation

### Installation
1. **Téléchargement** : Téléchargez le fichier `ConfigScan.zip` 
2. **Extraction** : 
   - Faites un clic droit sur le fichier `ConfigScan.zip`
   - Sélectionnez "Extraire vers..."
   - Choisissez le dossier de destination (ex: Bureau ou Documents)
3. **Lancement** : Naviguez vers le dossier extrait et double-cliquez sur `ConfigScan.exe` pour lancer l'application

## 📖 Utilisation

1. **Lancement** : Exécutez `ConfigScan.exe`
2. **Configuration** : Ajustez les paramètres de scan selon vos besoins
3. **Scan** : Cliquez sur "Démarrer le scan" pour lancer l'analyse
4. **Résultats** : Consultez les résultats dans l'interface ou exportez le rapport

**ConfigScan** - Votre solution complète pour l'analyse de configuration système Windows.
